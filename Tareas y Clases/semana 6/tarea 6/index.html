Selectores en CSS

¿Qué son los selectores en CSS?

Los selectores en CSS son patrones utilizados para seleccionar y aplicar estilos a los elementos HTML. Son una herramienta fundamental para diseñar y dar formato visual a las páginas web.

¿Para qué sirven?

Sirven para:

  • Seleccionar elementos específicos del documento HTML.
  • Aplicar reglas de estilo (colores, tamaños, márgenes, etc.).
  • Crear estructuras visuales dinámicas y coherentes.

Tipos de selectores más comunes:

  • Selector de tipo (o de etiqueta):
    Aplica estilos a todas las etiquetas de un tipo específico.
    p { color: blue; }
  • Selector de clase:
    Se aplica a los elementos que tengan una clase específica.
    .destacado { font-weight: bold; }
  • Selector de ID:
    Se aplica a un elemento único con un identificador.
    #titulo { font-size: 24px; }
  • Selector descendiente:
    Aplica estilos a un elemento dentro de otro.
    div p { color: green; }
  • Selector de hijo directo:
    Aplica estilos solo a los elementos hijos directos.
    ul > li { list-style: square; }
  • Selector universal (*):
    Selecciona todos los elementos del documento.
    * { margin: 0; padding: 0; }
  • Selectores de atributo:
    Seleccionan elementos por el valor de un atributo.
    input[type="text"] { border: 1px solid #000; }
  • Selectores de grupo:
    Permite aplicar el mismo estilo a varios elementos.
    h1, h2, p { font-family: Arial; }
